=========================================
      BOOKVERSE - PROJECT OVERVIEW
=========================================

1. TECHNOLOGY STACK
-----------------------------------------
• Core Framework: React 18+
• Language: TypeScript
• Build Tool/Bundler: Vite
• Styling: Tailwind CSS
• Animation Engine: Framer Motion (Used for complex 3D CSS transforms and scroll-linked UI)
• Routing: React Router DOM (v6)
• Icons: Lucide React

2. ARCHITECTURE & CORE SYSTEM FEATURES
-----------------------------------------
• Multi-Role System Architecture: Built-in support for "User", "Seller", and "Admin" roles via AuthContext.
• Polymorphic Component System: Custom UI elements (like `<Button asChild>`) allowing semantic HTML wrappers without losing styles.
• Custom Toast Notification Engine: Global, dismissible toast popups for robust, non-blocking form validation and system alerts.
• Advanced Responsive Grids: Complex DOM re-ordering for mobile vs desktop views (e.g., ensuring 3D animations stack correctly on mobile).
• Smart Pagination System: Algorithmic ellipsis pagination (e.g., 1 2 ... 5 6 7 ... 20) to elegantly handle infinite datasets without overflowing UI.
• Global State Management: React Context API used for Cart, Wishlist, and Authentication.
